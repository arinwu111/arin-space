# arinrin.space

一个持续生长的个人数字空间。它从一张个人主页开始，逐渐变成集合信息阅读、生活记录、思考工具与个人知识库的小世界。

🔗 [访问网站](https://www.arinrin.space)

## 网站里有什么

- **随时间变化的首页**：背景会在黎明、上午、正午、下午、黄昏和夜晚自动切换，并联动冥想小屋、信箱与日志牌的昼夜状态。图片来自 Unsplash，加载失败时自动回退到渐变背景。
- **AI 信息与工具**：聚合 AI 日报、快讯、精选和往期内容；支持 DeepSeek、OpenAI、Claude、通义千问、智谱、硅基流动和 Gemini。
- **每日记录**：晨间准备、晚间复盘、语音输入、AI 自动梳理和“今日手记”，同时提供历史记录与数据导出。
- **探索与决策**：五问法、慢想、北极星校准、三版奥德赛计划、主题周与主题月实验。
- **书房**：收藏、书影音清单、英语口语练习、本地播客 Markdown 阅读，以及基于第一性原理的中文论文解读。
- **健康与冥想**：训练、饮食、恢复和趋势记录，支持图片/PDF 报告分析；冥想小屋提供多种呼吸节奏与练习统计。
- **投资与自我观察**：行情、自选股、投资原则墙、决策冷静器，以及塔罗、八字等轻量自我觉察工具。
- **公共角落**：访客可以向信箱投递留言，审核后展示在留言墙；日志所有人可读，只有管理员可以发布和修改。

## 准备继续接入

- 冥想小屋的白噪音、环境音和轻音乐
- Apple Health 数据同步与更完整的健康趋势
- 摄影、AI 作品和设计素材展示
- 原创塔罗视觉素材
- 更稳定的跨设备数据同步

## 技术栈

| 层级 | 使用方式 |
| --- | --- |
| 前端 | 原生 HTML、CSS、JavaScript，无前端框架；响应式布局与 Web APIs |
| 部署与服务器 | Vercel，后端使用 Vercel Serverless Functions |
| 数据库 | Upstash Redis（兼容 Vercel KV 环境变量），用于留言、公开日志、部分健康数据和接口限流 |
| 本地数据 | localStorage、IndexedDB、File System Access API；个人记录和播客文稿优先留在用户设备 |
| AI | 自建统一 API 代理，支持多家模型与多模态输入；用户可以使用自己的 API Key |
| 外部服务 | Unsplash、AI HOT、市场行情、ICS 日历等 |

## 设计与数据原则

这是一个边使用、边发现需求、边改造的个人项目。私人数据默认保存在本地浏览器，公共内容才进入云端数据库；外部服务统一经过后端接口中转，并加入管理员验证、请求限流和基础安全检查。

项目仍在持续改造中。
